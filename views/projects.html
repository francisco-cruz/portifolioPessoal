Resume

A plataforma Smart Library tem o proposito de facilitar a interação dos alunos do IFRN Campus-Caicó com a Biblioteca, trazendo alternativas digitais para o acesso de seus recursos e sendo uma maneira competente de organizar o sistema. Com a aplicação é possível ver antes mesmo de sair de casa se as ferramentas que irá utilizar na Biblioteca estão disponíveis, como as salas e os computadores, podendo efetuar suas reservas e só aí locomover-se, desta forma os alunos terão o acesso ao ambiente de uma maneira organizada, planejada e de fato certa.

Cinemax é um Aplicativo fictício que se aproxima de uma Bilheteria Virtual. Em tempos de Pandemia, formas de remotas de fazer tarefas do dia-a-dia são sempre muito bem vindas.

Animals é um projeto desenvolvido com a finalidade de arrecadar Doações ou até mesmo novos lares para animaizinhs de Rua.

Smart Library

Sistema de Automação de Bibliotecas

Objetivo

Problemática: Os alunos se descolam de suas casas sem a certeza da disponibilidade dos recursos da Biblioteca do IFRN Campus-Caicó.


Solução: Desenvolver um sistema de automação dos recursos da Biblioteca, para que os alunos não precisem sair de casa para reservar horários em salas de estudo e/ou verificar a disponibilidade de Computadores

Fontes & Cores

A combinação das fontes Ubuntu e Opens Sans passa uma sensação de dinamicidade e modernidade, além de possuir uma leitura bastante confortável e agradável aos olhos. Essa associação é perfeita para sistemas que envolve aprendizado, pois instiga os usuários a continuarem lendo o conteúdo.

As cores da plataforma foram pensadas tendo como base o meio acadêmico em geral e a psicologia das cores, pois tons de azul trazem a sensação de calma, confiança e segurança exatamente o que os alunos precisam sentir em uma plataforma digital.

Layout & Navegação

Porque usamos essa navegação..

Foi usado de base a ferramenta Google Classroom, que os alunos da instituição fazem uso e estão familiarizados, por isso usar um Layout semelhante, ficaria mais facil para os usuários (os própios alunos) assimilarem as funcionalidades.

Considerações finais

Smart Library foi o meu primeiro projeto, foi onde de fato comecei a desenvolver páginas Web e ao mesmo tempo me descobrir como Designer de Interfaces. O processo de construção foi bastante gradual, desde as primeiras telinhas desalinhadas até a plataforma final. Desenvolver Smart Library sem dúvidas foi um abridor de portas para todos os desenvoldores, pois todos eramos inciantes, mas com esforço e muito estudo conseguimos entregar o melhor trabalho possível.
